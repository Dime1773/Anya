# CHANGELOG - Anya Distributor

## v1.11.1 (2024-01-15) - СУПЕР КЛАССНАЯ ВЕРСИЯ ⭐

### 🎉 Главные улучшения

#### 1. Параллельная проверка 15-20 устройств (⚡ КРИТИЧЕСКОЕ)
- **Проблема v1.10.8:** Синхронная проверка каждого IP по очереди (медленно)
- **Решение v1.11.1:** Асинхронная параллельная проверка 15-20 устройств одновременно
- **Результат:** Проверка 1200+ IP ускорена в 15-20 раз!
  - v1.10.8: ~15-20 минут
  - v1.11.1: ~60-90 секунд
- **Технология:** `asyncio` + `asyncio.Semaphore` + `asyncio.open_connection()`
- **Таймаут:** 2 сек на IP (до первого ответа, не полный ping)

**Код изменений (workers.py):**
```python
# БЫЛО (v1.10.8): синхронная проверка
for idx, branch in enumerate(self.branches, 1):
    server_status = "Да" if self._check_ip(ip) else "Нет"  # Ждёт каждый IP
    
# СТАЛО (v1.11.1): асинхронная параллельная
async def _check_all_async(self):
    semaphore = asyncio.Semaphore(self.max_concurrent)  # max=50
    tasks = [self._check_branch_async(b, i, semaphore) for i, b in enumerate(self.branches)]
    await asyncio.gather(*tasks)
```

#### 3. Исправление критической ошибки "Автомат ошибок" (🐛 BUG FIX)
- **Ошибка:** `'int' object has no attribute 'strip'`
- **Причина:** Значение "Префикс" из xlsx загружалось как `int` (1, 2, 3)
- **Решение:** Конвертируем ВСЕ значения в строку перед обработкой

**Код изменений (workers.py):**
```python
# БЫЛО (v1.10.8): ошибка когда значение int
prefix = record.get("Префикс", "").strip()  # ← Падает если int

# СТАЛО (v1.11.1): всегда работает
prefix = str(record.get("Префикс", "")).strip()  # ← Сначала конвертируем в str
ip = str(record.get("IP", "")).strip()
```
