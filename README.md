# Anya Distributor v1.11.1

## ✨ Новое в версии 1.11.1

### 1️⃣ Параллельная проверка 15-20 устройств
- Проверка подключения теперь выполняется параллельно для 15-20 устройств одновременно
- Проверка идёт до **первого ответа** (без ожидания полного ping)
- Значительно ускорило процесс проверки 1200+ IP адресов

### 2️⃣ Переименование кнопок логов
- Кнопка "Развернуть логи" → "**Свернуть логи**" (когда логи видны)
- Кнопка "Развернуть логи" (когда логи скрыты)
- Логи сворачиваются на нижней панели

### 3️⃣ Исправление ошибки "Автомат ошибок"
- **Исправлена ошибка**: `'int' object has no attribute 'strip'`
- Все значения теперь конвертируются в строки перед обработкой
- Полная переработка логики автоматической передачи

### 4️⃣ Новый функционал "Автомат ошибок"
Полностью переписана логика работы с `errors.xlsx`:

**Процесс:**
1. Нажимаете кнопку "Автомат ошибок"
2. Выбираете файлы или папки для передачи
3. Приложение автоматически:
   - Проверяет наличие файла `logs/errors.xlsx`
   - Читает подразделения и IP из файла
   - Асинхронно проверяет доступность каждого IP (15-20 параллельно)
   - При доступности IP - передаёт файлы на диск D
   - Удаляет успешно обработанные записи из xlsx
   - Продолжает со следующими подразделениями

**Структура errors.xlsx:**
```
Префикс | IP
--------|----------
AB01    | 10.0.1.5
AB02    | 10.0.2.5
AB03    | 10.0.3.5
```

## 📋 Основные возможности

### 1. Загрузка базы данных
- Нажмите "Загрузить БД"
- Приложение загружает `apteki.json` и создаёт `base.json`

### 2. Проверка подключения
- Выберите подразделения (отметьте галочки)
- Нажмите "Проверить подключение"
- **Параллельная проверка** 15-20 устройств одновременно
- Проверяется доступность на **порт 445 (SMB)**
- Результаты показываются в столбцах "Сервер", "Оператор", "Оператор 2"

### 3. Передача файлов
- Выберите подразделения
- Нажмите "Передача файлов"
- Выберите файлы или папки
- Выберите целевые IP (Сервер/ОП1/ОП2)
- Нажмите "Начать передачу"

### 4. Логи и ошибки
- **Вкладка "Логи"** - все действия пользователя
- **Вкладка "Ошибки"** - ошибки передачи с экспортом в Excel
- **Кнопка "Свернуть логи"** - скрывает консоль
- **Кнопка "Развернуть логи"** - показывает консоль

### 5. Автомат ошибок ⚡
- Нажмите "Автомат ошибок"
- Выберите файлы/папки для передачи
- Приложение автоматически:
  - Читает `logs/errors.xlsx`
  - Проверяет IP параллельно (15-20 за раз)
  - Передаёт файлы при наличии доступного IP
  - Удаляет успешные записи из xlsx

## 🐛 Исправленные ошибки

### v1.11.1 vs v1.10.8

1. ✅ **Ошибка "int object has no attribute 'strip'"**
   - Причина: значение из xlsx было int вместо str
   - Решение: все значения конвертируются в str перед обработкой

2. ✅ **Параллельная проверка**
   - Было: синхронная проверка (медленно)
   - Стало: асинхронная параллельная проверка 15-20 устройств

3. ✅ **Кнопка логов**
   - Было: "Развернуть логи" (всегда)
   - Стало: "Свернуть логи" (когда видны), "Развернуть логи" (когда скрыты)

4. ✅ **Автомат ошибок**
   - Было: непонятная логика
   - Стало: четкий процесс с асинхронной проверкой IP

### Асинхронность
- Используется `asyncio` для параллельной обработки
- `asyncio.Semaphore` ограничивает одновременные TCP подключения до 20
